看本篇之前，可以先了解clash for Android 的用法.

教程只有两步.

第一步 下载

我们可以到clash for Windows 的官方GitHub地址下载.（注意：需要翻墙）

如果没有翻墙，可以选择蓝奏云地址密码:h957.（这个版本太老了，记得下载完更新一下）

这是汉化版GitHub地址.

第二步 导入

类似clash for Android 的用法

最上方分别代表的意思：

  • 那个意为“粘贴文件”的图标——从剪切板粘贴
  • Download——下载
  • Update All——（配置）全部更新
  • Import——导入（.yaml或者.yml文件）

主页中文 他这个有点旧了

导入和clash for Android 的做法一样 在配置（Profiles）界面粘贴进链接（或者Import文件进行添加） 之后在代理（Proxies）界面选择一个能用的.
